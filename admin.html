📖 Cara Menggunakan Dashboard

                        
Panduan lengkap penggunaan portal admin portofolio ini.

                    


                    

                        
🚀 Alur Kerja

                        
    
                            
  • 1
    Tambah Portofolio — Pergi ke menu "Tambah Portofolio", isi semua field yang diperlukan, lalu klik "Simpan Portofolio".
    • 
                            
  • 2
    Langsung Publik! — Item Anda akan otomatis tampil di halaman portfolio publik tanpa langkah tambahan. Cukup refresh halaman portfolio.
    • 
                            
  • 3
    Kelola Item — Di menu "Daftar Item" Anda bisa melihat semua item yang sudah disimpan, filter per kategori, atau hapus item yang tidak diperlukan.
    • 
                            
  • 4
    Update Gambar — Letakkan file gambar baru di folder images/, lalu isi field "URL Gambar" dengan nama file, contoh: images/foto-baru.jpg.
    •

⚠️ Catatan Penting

                        

                            🌐 

                                Dengan GitHub Sync aktif, setiap item yang disimpan akan langsung ditulis ke file portfolio-data.json di repo GitHub Anda. Halaman portfolio publik membaca file ini — sehingga semua pengunjung dapat melihatnya dalam ~30-60 detik (waktu GitHub Pages rebuild).
                            

                        

                        

                            🔑 

                                Login ini bersifat client-side dan dirancang sebagai lapisan akses dasar. Jangan gunakan password penting yang sama dengan akun lain di sini.
                            

                        

                    

                


                
                

                    

                        
⚙️ GitHub Sync

                        
Hubungkan dashboard ke repositori GitHub agar portofolio langsung dipublikasikan ke semua pengunjung.

                    


                    

                        
🌐 Cara Kerja

                        

                            Setiap kali Anda menyimpan portofolio, admin akan otomatis menulis ke file portfolio-data.json di repo GitHub Anda via GitHub API.
                            File tersebut dibaca langsung oleh halaman portfolio publik — sehingga semua pengunjung dari device manapun bisa melihatnya, bukan hanya browser Anda saja.
                        

                    


                    

                        
🔧 Konfigurasi GitHub

                        

                            

                                

                                    
                                    
                                

                                

                                    
                                    
                                

                                

                                    
                                    
                                

                                

                                    
                                

                                

                                    
                                    
                                    
💡 Buat di: GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) → centang repo

                                

                            

                            

                            

                                
                            

                        

                    


                    

                        
📋 Cara Mendapat Personal Access Token

                        
    
                            
  • 1
    Login ke github.com → klik foto profil → Settings
    • 
                            
  • 2
    Scroll ke bawah → Developer settings → Personal access tokens → Tokens (classic)
    • 
                            
  • 3
    Klik Generate new token (classic), beri nama misal "Portfolio Admin", pilih expiry, centang repo (full repo access)
    • 
                            
  • 4
    Klik Generate token, salin tokennya sekarang (hanya tampil sekali!), lalu paste di field token di atas
    • 
                        

                        

                            🔒 
Token disimpan di localStorage browser Anda dan hanya dikirim ke api.github.com. Tidak ada server pihak ketiga. Jangan bagikan token ini ke siapapun.
