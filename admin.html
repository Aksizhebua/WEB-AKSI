📖 Petunjuk Menempel Kode

                        
    
                            
  • 1 Buka file portfolio.html di editor teks (VS Code, dll).
    • 
                            
  • 2 Cari baris komentar <!-- Portfolio Grid --> di dalam file.
    • 
                            
  • 3 Tempelkan kode kartu baru di dalam tag <div class="portfolio-grid">.
    • 
                            
  • 4 Untuk Featured, cari bagian <!-- Featured Project --> dan ganti isi .featured-card.
    • 
                            
  • 5 Simpan file dan refresh browser untuk melihat hasilnya.
    • 
                        

                    

                


                
                

                    

                        
📖 Cara Menggunakan Dashboard

                        
Panduan lengkap penggunaan portal admin portofolio ini.

                    


                    

                        
🚀 Alur Kerja

                        
    
                            
  • 1
    Tambah Portofolio — Pergi ke menu "Tambah Portofolio", isi semua field yang diperlukan, lalu klik "Simpan".
    • 
                            
  • 2
    Kelola Item — Di menu "Daftar Item" Anda bisa melihat semua item yang sudah disimpan, filter per kategori, atau hapus item yang tidak diperlukan.
    • 
                            
  • 3
    Ekspor ke Website — Pergi ke "Ekspor HTML", pilih jenis ekspor, salin kode HTML yang dihasilkan, lalu tempel ke file portfolio.html di posisi yang sesuai.
    • 
                            
  • 4
    Update Gambar — Letakkan file gambar baru di folder images/, lalu isi field "URL Gambar" dengan nama file, contoh: images/foto-baru.jpg.
    •

⚠️ Catatan Penting

                        

                            🔒 

                                Data portofolio disimpan secara lokal di browser Anda melalui localStorage. Data ini tidak akan hilang jika browser ditutup, namun tidak otomatis muncul di website publik — Anda perlu melakukan ekspor HTML dan menempelkannya ke file portfolio.html secara manual.
                            

                        

                        

                            🔑 

                                Login ini bersifat client-side (tersimpan di browser) dan dirancang sebagai lapisan akses dasar. Jangan gunakan password penting yang sama dengan akun lain di sini. Untuk keamanan lebih kuat, pertimbangkan migrasi ke platform dengan backend (seperti Netlify CMS, Sanity, dll) di masa mendatang.
